eJava
============

A [Servlet API](http://docs.oracle.com/javaee/6/api/javax/servlet/package-summary.html) Framework.

一个极轻量的JAVA框架，适合开发API，采用Maven。

How to use it:

    <dependency>
        <groupId>com.erlitech</groupId>
        <artifactId>ejava</artifactId>
        <version>1.1.7</version>
    </dependency>
